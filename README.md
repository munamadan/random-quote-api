# Random Quote API

A complete FastAPI system for managing and retrieving random quotes.

## Features

- **CRUD Operations**: Create, read, update, and delete quotes
- **Random Quote Generation**: Get random quotes with category filtering
- **Category Management**: Organize quotes by categories
- **RESTful API**: Fully REST-compliant endpoints
- **SQLite Database**: Lightweight database with sample data
- **Automatic Documentation**: Interactive API docs at `/docs`
